# distill-after-work

A reusable procedure (skill) for the distillation phase of the loop. The block of
`uses` / `score` / `votes` / `last_used` keys above is the **scorecard**: it turns this
vault into a rated *skills library*, not just memory.

## Steps

1. Split what you learned into atomic notes (one idea per page).
2. Add frontmatter and at least one `[[wikilink]]`.
3. Update `index.md`, `hot.md`, and `log.md`.
4. Run `brain check`.

## Scorecard

- The agent runs `brain skill use distill-after-work` after applying it.
- Anyone can run `brain skill rate distill-after-work <1-5> [note]` to vote on quality.
- `brain skill list` ranks the whole library by score, then by uses.
